# 数据归档设计实验

## 简介

本次实验目的是为了演示如何对“数据库大表”进行归档/月切/数据迁移，解决因数据量太大而导致的数据库性能下降的性能问题。

## 实验环境
 - Windows 11
 - Redis 5.0
 - MySQL 8.0
 - JDK 17

## 技术点
- 定时任务
- 事务
- 重试机制
- 分布式锁

## 使用方法
1. 准备好环境、依赖
2. 改改配置文件
3. 运行项目

## 适用场景
 - 适用于 单体架构系统/单体架构集群部署的场景
